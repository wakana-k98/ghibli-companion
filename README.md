# 🐾 Pup To-Do

A cute to-do list with a dog companion who needs walks and feeding. Completing tasks tagged "walk" or "feed" fills up your pup's meters and keeps them happy — neglect them and they get a little sad.

## Features

- 🦴 **Walk meter** — fills up as you complete "walk" tasks
- 🍖 **Feed meter** — fills up as you complete "feed" tasks
- A hand-drawn SVG dog that bounces when you finish a task, and looks sad/tired when both meters run low
- Mood messages that change based on how well-cared-for your pup is
- Tasks and pup stats are saved locally in your browser (`localStorage`) — no backend needed
- Single self-contained HTML file, no build step or dependencies
